## Simple Descriiption

### ✅ What is DE:BOOST

DE:Boost는 머신러닝 기술을 바탕으로 각 티어(Tier, 게임의 실력 지표가 되는 랭킹)별 행동 양식과 사용자별 최근 플레이 데이터를 분석하여, 해당 사용자의 부스팅 여부를 판단하고 신고할 수 있는 **AI 기반 웹 서비스**입니다.

개발 전, 플레이어의 전적 데이터 10만건을 사용하여 로그 데이터를 추출 한뒤 머신러닝을 통해 얻은 학습 모델을 기반으로 **사용자의 랭크를 예측할 수 있는 기술**을 구축하였습니다.

또한 의심 플레이어의 현재 랭크와 예측값으로 나온 랭크의 차이가 일정 수준을 넘을 경우 사용자에게 부정행위 방지 팀에게 즉각적으로 전달할 수 있는 **리포트를 제공**합니다.
